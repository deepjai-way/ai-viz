# 辐射汇聚图（Convergence）渲染规格

## 适用场景

- 知识源包含"多个因素共同导致某个结果"的因果/条件汇聚描述
- 或反向："一个源头产生多个影响/输出"的辐射扩散描述
- 需要回答"什么导致了这个结果？"或"这个东西产生了什么影响？"
- 典型知识源：成因分析、爆发条件、影响评估、生态图谱

## 布局规范

- **方向**：中心辐射式（center-radial），核心概念居中，条件/结果环绕
- **画布建议尺寸**：900×800 ~ 1200×1000px（正方形偏好，避免拉伸感）
- **中心区域**：画布几何中心偏上，留出标题空间
- **环绕节点分布**：均匀分布在中心的上下左右四方位

### 变体选择

| 变体 | 适用场景 | 布局特点 |
|------|---------|---------|
| 汇聚型（多→一） | 多条件促成一个结果 | 条件在外围，箭头指向中心 |
| 辐射型（一→多） | 一个源产生多种影响 | 源在中心，箭头指向外围 |
| 四象限型 | 条件恰好 4 个，需要对称美感 | 上下左右各一个 |
| 半环型 | 条件 3-5 个，需要紧凑 | 围成半弧环绕中心 |

### 复杂度分级

| 环绕节点数 | 策略 |
|-----------|------|
| 2-3 | 紧凑对称（radius=200px） |
| 4 | 标准四象限（radius=250px） |
| 5-6 | 宽松环绕（radius=300px） |
| > 6 | 建议拆分或分组后再辐射 |

### 间距参数

- 中心到环绕节点距离（radius）：200-300px
- 环绕节点之间无需直接间距约束（由 radius + 角度自动决定）
- 箭头长度：中心椭圆边缘 → 环绕节点边缘之间，留 20-40px 间隙

## 配色规范

### 核心原则

- 中心节点**视觉最突出**（accent 或 muted+粗边框）
- 环绕节点每个一色（区分不同条件/影响），或统一颜色（强调整体性）
- 主色 <= 3 种

### 配色方案

| 策略 | 适用场景 | 映射 |
|------|---------|------|
| 中心强调 | 结果最重要，条件次之 | 中心=accent/muted+粗框，环绕=primary/secondary 循环 |
| 条件强调 | 每个条件有独立语义 | 中心=muted（弱化），环绕=各色区分 |
| 统一色系 | 强调整体协同关系 | 中心=accent，环绕=同一 primary 色 |

### 默认映射（4 条件汇聚为例）

| 元素 | 色彩槽位 | 说明 |
|------|---------|------|
| 中心椭圆 | muted | 低饱和灰底，让箭头和标题成为焦点 |
| 条件 1（上） | primary | 蓝色 |
| 条件 2（右） | secondary | 绿色 |
| 条件 3（下） | accent | 黄色 |
| 条件 4（左） | danger | 红色 |

> 当环绕节点 > 3 且颜色克制原则冲突时，可只用 2 色交替（primary + secondary），中心用 accent 强调。

## 元素规范

### 中心节点

| 属性 | 规范 |
|------|------|
| 形状 | 椭圆（ellipse）— 表达"聚合"意象 |
| 尺寸 | 宽 280-320px，高 120-160px |
| 背景 | muted 或 accent（取决于配色策略） |
| 边框 | 2-3px，黑色 |
| 内容 | 主标题（22-24px）+ 副标题（16-18px） |

### 环绕节点

| 属性 | 规范 |
|------|------|
| 形状 | 圆角矩形 |
| 尺寸 | 宽 250-300px，高 110-140px |
| 背景 | 对应色彩槽位 |
| 内容 | 标题（20-22px）+ 描述文字（14px，浅灰色） |
| 描述 | 1-2 行简短说明，控制在 20 字以内 |

### 箭头

| 方向 | 样式 | 说明 |
|------|------|------|
| 汇聚型 | 环绕→中心，实线箭头，sw=3 | 表达"条件促成结果" |
| 辐射型 | 中心→环绕，实线箭头，sw=3 | 表达"源产生影响" |

箭头规范：
- 起点：环绕节点朝向中心的边缘
- 终点：中心椭圆朝向该环绕节点的边缘
- 不标注文字（简洁优先），除非箭头需要区分不同类型的关系

### 标题

- 位置：画布顶部居中
- 字号：32-36px
- 与中心节点保持 80-120px 距离

## 最小完整示例结构

### 汇聚型（4 条件→1 结果）

```
             [标题]

        [条件1]
            ↘
[条件4] →  (结果)  ← [条件2]
            ↗
        [条件3]
```

### 辐射型（1 源→4 影响）

```
             [标题]

        [影响1]
            ↗
[影响4] ←  (源)  → [影响2]
            ↘
        [影响3]
```

## 质量自检补充项

- [ ] 中心节点位于视觉中心（画布几何中心附近）
- [ ] 环绕节点分布均匀，无明显倾斜
- [ ] 所有箭头方向一致（全部向内或全部向外）
- [ ] 箭头不穿越其他节点
- [ ] 中心与环绕之间留有足够间隙（不粘连）
- [ ] 环绕节点尺寸一致（视觉平衡）

## 实战案例

- 《生成式AI爆发的四大条件》：4 条件汇聚型，中心椭圆"生成式AI爆发"，四方位分布数据/算力/模型/产品
